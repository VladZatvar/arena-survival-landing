# Інструкція з оновлення проєкту

## 1. Підготовка до оновлення

Перед оновленням потрібно:

- переконатися, що локальний репозиторій не містить незбережених змін
- перевірити сумісність нових змін із поточною структурою проєкту
- за потреби створити резервну копію

## 2. Процес оновлення

1. Отримати актуальні зміни:

```bash
git pull
```
