# Інструкція з розгортання проєкту

## 1. Загальна інформація

Проєкт `arena-survival-landing` є статичним вебсайтом, створеним для презентації бакалаврської роботи.

Технології проєкту:

- HTML
- CSS
- JavaScript
- npm для службових інструментів
- GitHub Pages для публікації

## 2. Архітектура проєкту

Структура проєкту включає:

- вебклієнт у вигляді статичних файлів `index.html`, `style.css`, `script.js`
- документацію в папці `docs`
- автоматично згенеровану документацію JSDoc у папці `docs/api`

У проєкті не використовуються:

- application server
- база даних
- окреме файлове сховище
- кешуючі сервіси

## 3. Вимоги до середовища

Для розгортання необхідно:

- Windows 10/11 або інша сучасна ОС
- Git
- Node.js та npm
- браузер
- доступ до GitHub

## 4. Порядок розгортання

1. Клонувати репозиторій:

```bash
git clone https://github.com/VladZatvar/arena-survival-landing.git
```
