# Інструкція з генерації документації

## Призначення

У проєкті використовується JSDoc для автоматичної генерації документації з JavaScript-коду.

Документація генерується на основі JSDoc-коментарів, доданих до функцій у файлі `script.js`.

## Необхідні умови

Перед генерацією документації потрібно:

1. Встановити Node.js та npm
2. Встановити залежності проєкту командою:

```bash
npm install
```
