# Arena Survival Landing

## Опис

Односторінковий лендінг для презентації бакалаврської роботи на тему:

**Ігровий застосунок жанру виживання на арені з процедурною зміною ігрових подій**  
**Game Application in the Arena Survival Genre with Procedurally Changing Game Events**

## Мета

Представити тему бакалаврської роботи у форматі веб-лендінгу з дотриманням базових вимог до SEO, доступності, адаптивності та роботи з Git.

## Структура

- `index.html` — головна сторінка
- `style.css` — стилі сайту
- `robots.txt` — правила для пошукових систем
- `sitemap.xml` — карта сайту
- `images/` — зображення
- `.gitignore` — виключення службових файлів

## Розгортання

Планується публікація через GitHub Pages.

## Документування коду

У проєкті використовується файл `script.js`, який містить JavaScript-функції для інтерактивної поведінки лендінгу.

Для документування коду використовується формат JSDoc.  
Автоматична генерація документації виконується за допомогою інструмента `jsdoc`.

### Генерація документації

```bash
npm run docs:clean
npm run docs
```

## Документація розгортання та підтримки

У проєкті підготовлено окрему документацію для розгортання, оновлення та резервного копіювання.

### Файли документації

- `docs/deployment.md` — інструкція з розгортання проєкту
- `docs/update.md` — інструкція з оновлення проєкту
- `docs/backup.md` — інструкція з резервного копіювання

### Допоміжні скрипти

У папці `docs/scripts` підготовлено службові bat-файли:

- `dev.bat` — підготовка локального середовища розробки
- `prod.bat` — перевірка перед production-оновленням
- `backup.bat` — створення резервної копії основних файлів проєкту

### Особливості розгортання

Проєкт є статичним вебсайтом і не використовує:

- application server
- базу даних
- окреме файлове сховище
- кешуючі сервіси

Публікація виконується через GitHub Pages.
